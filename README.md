# 1. Project Overview (프로젝트 개요)
우리 AES 서비스는 사용자의 기술 스택과 이력을 정밀하게 분석하여 최적의 채용 정보를 매칭해 주는 AI 기반 취업 지원 플랫폼이다.
사회초년생(취업준비생)들에게는 특히나 복잡한 채용 시장에서 개인화된 데이터를 바탕으로 구직 효율을 극대화하는 것을 목표로 한다.
